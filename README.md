CircleIndicator
===============
a lightweight viewpager indicator like in nexus 5 launcher 


Usage
--------
```xml
	<me.relex.circleindicator.CircleIndicator
        android:layout_width="fill_parent"
        android:layout_height="40dp"/>
```

#####Properties:

*`app:ci_width`
*`app:ci_height`
*`app:ci_margin`
*`app:ci_animator`
*`app:ci_drawable`
